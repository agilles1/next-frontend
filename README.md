## Features 
- Create multiple auditions with unique candidates
- Check-in new candidates and auto assign candidate numbers 
- Move candidates through rooms automatically to maximize efficiency
 
## Tech Stack
This web app makes use of the following:
- Ruby 
- Rails - MVC Web Framework
- SQLite3  - Database
- React.js
- Bootstrap
- Material-UI

## Installing

- Clone this repo to your local machine `git clone <this-repo-url>`
- run `yarn install` to install required dependencies
- run `yarn start` to run the server.
